Application Startup Error

Dashboard Penilaian Kinerja Pegawai Dinas Sosial failed to start properly. This might be due to a server startup issue or missing dependencies.

Troubleshooting Steps:

  • Check if another instance of the application is running
  • Ensure you have write permissions to the user data directory
  • Try running the application as administrator (Windows)
  • Check if your antivirus software is blocking the application
  • Verify that the application files are not corrupted

If the problem persists:

  • Check the application logs in the user data directory
  • Try reinstalling the application
  • Contact support with error details
